Requisitos do Sistema

Antes de começar a instalação, certifique-se de que seu ambiente atende aos seguintes requisitos:

Componente Versão Mínima Recomendada
Node.js 16.x 18.x ou superior
npm 8.x 9.x ou superior
Git 2.x Última versão
Nota: Esta aplicação é construída com Next.js, que pode ser implantada como uma aplicação Node.js ou exportada como arquivos estáticos.

Configuração do Ambiente

Instalação do Node.js e npm

1 Instale o Node.js e npm

Baixe e instale o Node.js a partir do site oficial. Isso também instalará o npm.

Verifique a instalação com:

node --version
npm --version

Instalação do Git

2 Instale o Git

Baixe e instale o Git a partir do site oficial.

Verifique a instalação com:

git --version

Passos para Instalação

Clonando o Repositório

1 Clone o repositório

git clone [URL_DO_REPOSITÓRIO]
cd street-dog-coin

Instalando Dependências

2 Instale as dependências

npm install

Configuração de Variáveis de Ambiente

3 Configure as variáveis de ambiente

Crie um arquivo .env.local na raiz do projeto com as seguintes variáveis:

# Credenciais de administrador
ADMIN_USERNAME=seu_usuario_admin
ADMIN_PASSWORD=sua_senha_admin
NEXT_PUBLIC_ADMIN_USERNAME=seu_usuario_admin_publico
NEXT_PUBLIC_ADMIN_PASSWORD=sua_senha_admin_publica

# Configurações da aplicação
NEXT_PUBLIC_SITE_URL=https://seu-dominio.com

Executando em Desenvolvimento

4 Inicie o servidor de desenvolvimento

npm run dev

A aplicação estará disponível em http://localhost:3000

Construção para Produção

5 Construa a aplicação para produção

npm run build

Executando em Produção Localmente

6 Inicie o servidor de produção

npm start

A aplicação estará disponível em http://localhost:3000

Implantação na Vercel

A Vercel é a plataforma recomendada para aplicações Next.js. Siga estes passos para implantar na Vercel:

1 Crie uma conta na Vercel

Acesse https://vercel.com/signup e crie uma conta se ainda não tiver uma.

2 Instale a CLI da Vercel (opcional)

npm install -g vercel

3 Implante usando a CLI (opcional)

vercel

Siga as instruções para fazer login e configurar o projeto.

4 Implante usando o Dashboard da Vercel

  1. Faça login no Dashboard da Vercel
  2. Clique em "Add New" e selecione "Project"
  3. Importe seu repositório Git
  4. Configure as variáveis de ambiente (veja a seção abaixo)
  5. Clique em "Deploy"
Dica: A Vercel detectará automaticamente que é um projeto Next.js e aplicará as configurações recomendadas.

Variáveis de Ambiente

Configure as seguintes variáveis de ambiente na Vercel:

Variável Descrição Exemplo
ADMIN_USERNAME Nome de usuário do administrador (servidor) admin123
ADMIN_PASSWORD Senha do administrador (servidor) senha_segura_123
NEXT_PUBLIC_ADMIN_USERNAME Nome de usuário do administrador (cliente) admin123
NEXT_PUBLIC_ADMIN_PASSWORD Senha do administrador (cliente) senha_segura_123
NEXT_PUBLIC_SITE_URL URL do site em produção https://seu-dominio.com
Atenção: Nunca compartilhe suas credenciais de administrador. Use senhas fortes e únicas para cada ambiente.

Solução de Problemas Comuns

Problemas de Build

Problema: Falha no build com erros de módulos não encontrados.
Solução: Verifique se todas as dependências estão instaladas corretamente. Execute npm install novamente.
Problema: Erros de variáveis de ambiente.
Solução: Certifique-se de que todas as variáveis de ambiente necessárias estão definidas no arquivo .env.local ou no ambiente do servidor.

Problemas de Autenticação

Problema: Não consegue fazer login como administrador.
Solução: Verifique se as variáveis de ambiente ADMIN_USERNAME, ADMIN_PASSWORD, NEXT_PUBLIC_ADMIN_USERNAME e NEXT_PUBLIC_ADMIN_PASSWORD estão configuradas corretamente.

Problemas de Armazenamento

Problema: Dados não estão sendo salvos corretamente.
Solução: Em ambientes serverless como a Vercel, o armazenamento em arquivo não é persistente. Considere migrar para um banco de dados como MongoDB, PostgreSQL ou usar o Vercel KV.

Problemas de Performance

Problema: Animações lentas em dispositivos móveis.
Solução: Ative o "Modo Leve" clicando no botão flutuante no canto inferior direito da tela.
